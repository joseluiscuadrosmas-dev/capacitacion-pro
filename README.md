# Capacitación Pro

Aplicación web para preparar el examen de Capacitación Profesional de Transporte.

## Sprint 1 - Foundation

Este paquete contiene una base estable para trabajar en la rama `develop`.

## Archivos principales

- `index.html`: aplicación v1.1 estable.
- `preguntas.js`: banco de preguntas cargado por la aplicación.
- `data/raw/`: ficheros originales `cm1A.txt` a `cm1H.txt`.
- `data/preguntas.js`: copia del banco de preguntas preparada para futura modularización.
- `docs/`: documentación inicial del proyecto.

## Cómo probar

1. Abrir `index.html` con doble clic.
2. Confirmar que carga el asistente o el plan de estudio.
3. Hacer un test y verificar la puntuación: correcta `+1`, errónea `-0,5`, no contestada `0`.

## Importante

La versión pública vive en `main`. Esta versión debe subirse primero a `develop` para pruebas.
